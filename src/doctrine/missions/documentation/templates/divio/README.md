# Divio Content Templates

Content scaffolds following the Divio 4-type documentation system. Each template
provides the canonical structure for one documentation type.

| Template | Type | Orientation |
|----------|------|-------------|
| `tutorial-template.md` | Tutorial | Learning-oriented, teaches beginners step-by-step |
| `howto-template.md` | How-To | Task-oriented, solves specific problems |
| `reference-template.md` | Reference | Information-oriented, describes APIs |
| `explanation-template.md` | Explanation | Understanding-oriented, explains architecture and rationale |
