2. Open the dashboard in a browser (or use `curl` for JSON endpoints). Verify:

   | Health status | `GET /api/health` | Status "ok"; sync section present |

4. Check the browser console for JavaScript errors (or use `curl` to verify
   response shapes match expectations).

5. Stop the dashboard after the test:
   ```bash
   curl -s -X POST http://localhost:8899/api/shutdown
   ```
