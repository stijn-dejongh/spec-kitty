# WP01 — Auth probe and `AuthStatus` extension

## Scope

Add the probe module and extend the `AuthStatus` enum. Do NOT modify `_evaluate_uncached` in this WP — that wiring lands in WP03.

## Files

- `src/specify_cli/readiness/coordinator.py` — extend `AuthStatus` only.
- `src/specify_cli/readiness/auth.py` — new module.
- `tests/readiness/test_auth_probe.py` — new tests.

## Implementation outline

1. In `coordinator.py`, add 4 new members to `AuthStatus`:
   - `AUTHENTICATED = "authenticated"`
   - `LOGGED_OUT_IN_TEAMSPACE = "logged_out_in_teamspace"`
   - `NOT_IN_TEAMSPACE = "not_in_teamspace"`
   - `UNKNOWN = "unknown"`
   Preserve `NOT_CHECKED` and `DISABLED` as backward-compat values.

2. Create `src/specify_cli/readiness/auth.py` with:
   ```python
   def probe_auth_status(*, repo_root: Path | None = None) -> tuple[AuthStatus, str | None]:
       """Return (status, teamspace_handle_or_None). Local signals only.
       Wraps all exceptions and returns (UNKNOWN, None) on failure.
       Resolution order:
         1. TokenManager.is_authenticated -> AUTHENTICATED
         2. detect_logged_out_with_connected_teamspace() returns non-None
            -> LOGGED_OUT_IN_TEAMSPACE
         3. else -> NOT_IN_TEAMSPACE
       """
   ```

3. Test cases in `test_auth_probe.py`:
   - `test_authenticated_session` — mock `TokenManager.is_authenticated = True` → `(AUTHENTICATED, None)`.
   - `test_logged_out_in_teamspace` — mock TM authenticated=False, helper → `"acme"`. Expect `(LOGGED_OUT_IN_TEAMSPACE, "acme")`.
   - `test_not_in_teamspace` — mock TM authenticated=False, helper → `None`. Expect `(NOT_IN_TEAMSPACE, None)`.
   - `test_unknown_on_token_manager_failure` — `get_token_manager` raises. Expect `(UNKNOWN, None)`.
   - `test_unknown_on_helper_failure` — helper raises. Expect `(UNKNOWN, None)`.
