# WP01 introduces no new canonical terms

WP01 is a type-safety infrastructure work package. It adds type stubs, fixes redundant casts,
removes stale `type: ignore` comments, and documents bug evidence. No new canonical domain
terms are introduced.

See `spec.md` Domain Language section for the canonical terms this mission reinforces —
all pre-existing terms (structural debt, deliberate linearity, pipeline-shape, etc.) are
introduced or reinforced by other WPs in this mission.
